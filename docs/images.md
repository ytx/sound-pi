# 画像アセット仕様

## ディレクトリ構成

```
py/assets/
├── fonts/
│   └── SpaceMono-Italic.ttf    # UI フォント
├── images/                      # 画像アセット（透過 PNG）
│   ├── vu_bg.png
│   ├── vu_needle.png
│   ├── vu_pivot.png
│   ├── dual_vu_bg.png
│   ├── dual_vu_needle.png
│   ├── spectrum_bar.png
│   ├── icon_vu_meter.png
│   ├── icon_dual_vu_meter.png
│   ├── icon_spectrum.png
│   ├── icon_mixer.png
│   ├── icon_bluetooth.png
│   └── icon_wifi.png
└── generate_assets.py           # Pillow による自動生成スクリプト
```

## 共通仕様

- フォーマット: **PNG (RGBA)**、全てアルファチャンネル付き
- 色空間: sRGB
- 背景: 透過 (`(0, 0, 0, 0)`)、vu_bg のみ不透明黒 `(0, 0, 0, 255)`
- ロード: `pygame.image.load().convert_alpha()` でキャッシュ (`ui/assets.py`)

### vu_needle.png — 針

| 項目 | 値 |
|------|-----|
| サイズ | 20 x 210 px |
| ピボット位置 | 画像下端中央 (10, 205) |
| 初期方向 | 上向き (90°) |

**描画要素:**

- テーパー四角形: 底幅 6px → 先端幅 2px、`(240, 240, 240, 230)`
- 中心ライン: 1px 白
- 先端: 赤三角 `(200, 50, 50)`

**回転方法:** `pygame.transform.rotate(needle, angle_deg - 90)` で回転し、`get_rect(center=pivot)` で中心補正。元画像から毎フレーム回転（累積回転しない）。

### vu_pivot.png — ピボットカバー

| 項目 | 値 |

**重ね順:** `vu_bg` → peak needle (line) → main needle (image) → `vu_pivot`

## スペクトラム

### spectrum_bar.png — バーグラデーションテクスチャ

| 項目 | 値 |
|------|-----|
| サイズ | 1 x 240 px |
| 用途 | 各バンドバーにスケーリングして blit |

**色グラデーション (上→下):**

| Y 範囲 | 対応レベル | 色 |
|--------|-----------|-----|
| 0–36 px | 85–100% | 赤 `(200, 0, 0)` |
| 36–96 px | 60–85% | 黄 `(200, 200, 0)` ← 緑からの補間 |
| 96–240 px | 0–60% | 緑 `(0, 200, 0)` |

**使用方法:** `pygame.transform.scale(tex, (BAR_W, 240))` でバー幅にスケーリング。各バンドは `surface.blit(tex, dest, src_rect)` で下から bar_h ピクセル分をクリップ表示。

## 画像の差し替え

手描きや外部ツールで作成した PNG に差し替え可能。条件:

1. **サイズを合わせる** — 上記の各サイズを守る
2. **透過 PNG (RGBA)** — アルファチャンネル必須
3. **針画像の向き** — 上向き (90°) で保存、ピボットは画像下端中央
4. ファイル名を同じにして `py/assets/images/` に配置

## 再生成

```bash
cd ~/git/sound-pi
source venv/bin/activate
python3 py/assets/generate_assets.py
```

Pillow (`pip install Pillow`) が必要。フォントは `py/assets/fonts/SpaceMono-Italic.ttf` を使用。

## 画像作成依頼用ガイド

以下は、デザイナーに画像作成を依頼する際にそのまま渡せる情報です。

### プロジェクト概要

Raspberry Pi に接続した **3.5インチ SPI LCD (480x320 ピクセル)** で動作するオーディオデバイスの UI 画像です。黒背景の画面にオーバーレイして使います。

### 表示環境

- 画面サイズ: **480 x 320 px**
- 画素密度: 約 165 ppi（3.5インチ）
- 背景色: **黒 (`#000000`)**
- UI のアクセントカラー: シアン `#00C8C8`、緑 `#00C800`、黄 `#C8C800`、赤 `#C80000`

### 共通の技術要件

| 項目 | 要件 |
|------|------|
| フォーマット | **PNG (RGBA)** — 全てアルファチャンネル付き |
| 色空間 | sRGB |
| 背景 | **透過** (vu_bg.png のみ不透明黒) |
| アンチエイリアス | あり（LCD の解像度が低いため、エッジを滑らかに） |

### 必要な画像一覧

#### VU メーター画面 — 全画面に 1 つのアナログメーター

| ファイル | サイズ (px) | 内容 |
|---------|------------|------|
| `vu_bg.png` | 480 x 320 | メーター盤面（目盛り・円弧・dBラベル）。背景は**不透明黒**。針は含めない |
| `vu_needle.png` | 20 x 210 | メーターの針。**上向き (12時方向)** で描く。ピボット (回転中心) は画像の**下端中央**。先端に赤いアクセント |
| `vu_pivot.png` | 30 x 30 | 針の根元を隠す丸い装飾カバー |

**vu_bg.png の詳細:**
- ピボット中心は画面座標 **(240, 280)**（画面下部寄り）
- 円弧の角度は **140° (左端) → 40° (右端)**、スイープ角 100°
- 目盛りの色: 0〜60% = 緑、60〜85% = 黄、85〜100% = 赤
- dB ラベル: -20, -10, -5, 0, +3, +6, +10

**vu_needle.png の注意:**
- プログラムで回転して使う（画像自体は常に上向きで保存）
- 底辺幅 6px → 先端幅 2px のテーパー形状
- 半透明で盤面が少し透けるとリアル

### デザインのトーン

- **ダーク基調** — 黒背景に映えるデザイン
- **計器・メカっぽい雰囲気** — アナログ VU メーターのレトロ感
- テキストや数値はプログラムで描画するので、画像に含めなくてよい（vu_bg の目盛りラベルは除く）

### 納品形式

- ファイル名を上記の通りにして、まとめて ZIP
- 可能であれば編集用のソースファイル（PSD, AI, Figma など）も
