# 画像アセット仕様

## ディレクトリ構成

```
py/assets/
├── fonts/
│   └── SpaceMono-Italic.ttf    # UI フォント
├── images/                      # 画像アセット（透過 PNG）
│   ├── vu_bg.png
│   ├── vu_needle.png
│   ├── vu_pivot.png
│   ├── dual_vu_bg.png
│   ├── dual_vu_needle.png
│   ├── spectrum_bar.png
│   ├── icon_vu_meter.png
│   ├── icon_dual_vu_meter.png
│   ├── icon_spectrum.png
│   ├── icon_mixer.png
│   ├── icon_bluetooth.png
│   └── icon_wifi.png
└── generate_assets.py           # Pillow による自動生成スクリプト
```

## 共通仕様

- フォーマット: **PNG (RGBA)**、全てアルファチャンネル付き
- 色空間: sRGB
- 背景: 透過 (`(0, 0, 0, 0)`)、vu_bg のみ不透明黒 `(0, 0, 0, 255)`
- ロード: `pygame.image.load().convert_alpha()` でキャッシュ (`ui/assets.py`)

### vu_needle.png — 針

| 項目 | 値 |
|------|-----|
| サイズ | 20 x 210 px |
| ピボット位置 | 画像下端中央 (10, 205) |
| 初期方向 | 上向き (90°) |

**描画要素:**

- テーパー四角形: 底幅 6px → 先端幅 2px、`(240, 240, 240, 230)`
- 中心ライン: 1px 白
- 先端: 赤三角 `(200, 50, 50)`

**回転方法:** `pygame.transform.rotate(needle, angle_deg - 90)` で回転し、`get_rect(center=pivot)` で中心補正。元画像から毎フレーム回転（累積回転しない）。

### vu_pivot.png — ピボットカバー

| 項目 | 値 |
|------|-----|
| サイズ | 30 x 30 px |

**描画要素:**

- 外側リング: `(50, 50, 55)` fill、`(120, 120, 120)` outline、線幅 2
- 内側ドット: `(80, 80, 85)`
- ハイライト: `(100, 100, 105, 80)` 半透明楕円

**重ね順:** `vu_bg` → peak needle (line) → main needle (image) → `vu_pivot`

## スペクトラム

### spectrum_bar.png — バーグラデーションテクスチャ

| 項目 | 値 |
|------|-----|
| サイズ | 1 x 240 px |
| 用途 | 各バンドバーにスケーリングして blit |

**色グラデーション (上→下):**

| Y 範囲 | 対応レベル | 色 |
|--------|-----------|-----|
| 0–36 px | 85–100% | 赤 `(200, 0, 0)` |
| 36–96 px | 60–85% | 黄 `(200, 200, 0)` ← 緑からの補間 |
| 96–240 px | 0–60% | 緑 `(0, 200, 0)` |

**使用方法:** `pygame.transform.scale(tex, (BAR_W, 240))` でバー幅にスケーリング。各バンドは `surface.blit(tex, dest, src_rect)` で下から bar_h ピクセル分をクリップ表示。

## 画像の差し替え

手描きや外部ツールで作成した PNG に差し替え可能。条件:

1. **サイズを合わせる** — 上記の各サイズを守る
2. **透過 PNG (RGBA)** — アルファチャンネル必須
3. **針画像の向き** — 上向き (90°) で保存、ピボットは画像下端中央
4. ファイル名を同じにして `py/assets/images/` に配置

## 再生成

```bash
cd ~/git/sound-pi
source venv/bin/activate
python3 py/assets/generate_assets.py
```

Pillow (`pip install Pillow`) が必要。フォントは `py/assets/fonts/SpaceMono-Italic.ttf` を使用。
